# 🧬 Asthma Prediction System

## 📌 Overview
Machine Learning model to predict asthma using clinical and demographic data.

## ⚙️ Models Used
- Random Forest (Best)
- XGBoost
- Neural Network

## 📊 Results
- Accuracy: 97%
- AUC: 0.98

## 🔍 Explainability
Used SHAP to interpret model predictions.
