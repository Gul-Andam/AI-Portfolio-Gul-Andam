# 📚 Comparative Study on Mental Health Detection

## 📌 Overview
Analyzed 17 research papers on sentiment-based mental health detection.

## 🔍 Key Findings
- Transformers outperform traditional ML
- Multimodal data is underused

## 📊 Evaluation Criteria
- Technique Advancement
- Dataset Quality
- Innovation
- Accuracy
- Modality Richness

## 🚀 Contribution
Proposed future direction for AI-based mental health systems.
