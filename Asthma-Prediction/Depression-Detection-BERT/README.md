# 🧠 Depression Detection using BERT

## 📌 Overview
NLP-based system to detect depression from social media posts.

## ⚙️ Model
- BERT Transformer
- Sentiment Analysis
- Emotion Detection

## 📊 Features
- Temporal tracking
- User behavior analysis

## 📈 Results
- Accuracy: 78% (can be improved)
